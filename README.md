# Saymara B. Ryon — Simply Be! YourSELF

Improved, redesigned website for **saymararyon.com** (H.M. Iulia Ioana / Saymara B. Ryon) — a holistic coach, art-therapist, TRE® facilitator and founder of the **ReUnity MATRIX®** method.

A fast, fully static, mobile-first site in Romanian, rebuilt from the original WordPress/Elementor site with a bespoke "flight rings" visual identity (gilded circular imagery, Poppins (titles) + Karla (body), light/warm-white background with gold accents).

## Pages (19)

- `index.html` — home
- `despre.html` — about + all methods (Coaching, Groups, Mentoring, TRE®, Resonanz®, Aspectologie®, Art-Therapy, POY®, Mandale de Lumină, Fosfenism)
- `cursuri-programe.html` + 3 course pages (Cartea Vieții Conștiente, TRE® 6 săptămâni, minicurs facilitatori)
- `reunity-matrix.html` — the ReUnity MATRIX® method
- `povesti-de-succes.html` — testimonials
- `portofoliu.html` — portfolio
- `blog.html` + 4 articles
- `galerie.html` — full photo gallery (every asset)
- `contact.html`
- Legal: `termeni-si-conditii.html`, `politica-cookies.html`, `politica-de-confidentialitate.html`

## Images

All **92 images** from the original site are carried over (optimized from ~100 MB to ~13 MB) plus 6 brand portraits, and every one is referenced on the site.

## Build

Header/footer/newsletter chrome is shared. Pages are assembled from `_build/src/*.html` fragments by `_build/wrap.ps1`; the gallery is generated by `_build/gen-gallery.ps1`. Both scripts must be saved **with a UTF-8 BOM** (PowerShell 5.1 requirement for the Romanian diacritics).

Static — no build step needed to deploy; served as-is on GitHub Pages.
